# 三档阅读计划

根据你的时间和目的，选择合适的阅读深度。

## 速读（~30分钟）

**适合场景**：
- 决定要不要读这本书
- 快速了解一个新领域
- 时间有限但需要核心信息

**流程**：
1. `/book-scan` — 全局扫描（10分钟）
2. `/book-card` — 知识卡片（10分钟）
3. `/book-export` — 导出笔记（可选）

**输出**：一张知识卡片 + 阅读策略建议

---

## 标准精读（~2小时）

**适合场景**：
- 想真正理解这本书
- 需要应用书中的方法论
- 准备写读书笔记或分享

**流程**：
1. `/book-scan` — 全局扫描（10分钟）
2. `/book-chapter` — 逐章拆解核心章节（每章10分钟，约3-5章）
3. `/book-quiz` — 苏格拉底质询（每章5分钟）
4. `/book-feynman` — 费曼输出（15分钟）
5. `/book-card` — 知识卡片（10分钟）
6. `/book-export` — 导出笔记
7. `/book-review` — 定期复盘（读完后随时可用）

**输出**：完整章节分析 + 理解评分 + 知识卡片 + 笔记文件

---

## 深度研究（不限时）

**适合场景**：
- 学术研究需要深度理解
- 要写书评或论文
- 要建立某个领域的知识体系

**流程**：
1. `/book-scan` — 全局扫描
2. `/book-chapter` — 逐章拆解所有章节
3. `/book-quiz` — 每章质询
4. `/book-feynman` — 每章费曼输出
5. `/book-critique` — 批判性审视
6. `/book-card` — 知识卡片
7. `/book-cross` — 跨书连接（读完3+本后）
8. `/book-export` — 导出完整笔记
9. `/book-review` — 定期复盘（读完后随时可用）

**输出**：完整书籍分析 + 批判性评价 + 跨书综合 + 笔记文件

---

## 如何选择

| 你的情况 | 推荐档位 |
|---------|---------|
| 刚拿到一本书，不知道值不值得读 | 速读 |
| 要考试/写论文/实际应用 | 标准精读 |
| 要深入研究某个领域 | 深度研究 |
| 时间只有30分钟 | 速读 |
| 时间有2小时 | 标准精读 |
| 时间不限 | 深度研究 |
