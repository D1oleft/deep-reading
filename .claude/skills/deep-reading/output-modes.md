# 输出模式说明

## 三种模式

### 极简模式（手机）
- 3-5 行核心结论
- 无格式符号
- 适合碎片时间阅读
- 手机端自动触发

### 标准模式（电脑端）
- 完整结构化分析
- 使用格式符号（━━━）
- 适合深度学习
- 默认模式

### 研究模式（写论文/书评）
- 标准模式 + 批判审视 + 跨书连接
- 最详细的分析
- 适合学术用途
- 用户说"详细分析"触发

## 自动切换

系统根据以下条件自动选择模式：
- 手机端 → 极简模式
- 电脑端 → 标准模式
- 用户说"详细/深入/研究" → 研究模式

## 模式对比

| 内容 | 极简 | 标准 | 研究 |
|------|------|------|------|
| 扫描结果 | 3行 | 完整 | 完整 |
| 章节拆解 | 5行 | 完整 | 完整+批判 |
| 知识卡片 | 5行 | 完整 | 完整 |
| 批判审视 | 无 | 可选 | 自动 |
| 跨书连接 | 无 | 可选 | 自动 |
