# 定期复盘

## 功能

读完一本书后，定期复盘检验记忆和理解，防止遗忘。

## 复盘时机

- **读完1天后**：快速复盘（5分钟）
- **读完1周后**：深度复盘（15分钟）
- **读完1个月后**：应用复盘（检验实际应用）

## 复盘方式

### 方式一：快速复盘（1天后）

```
📚 复盘：《书名》（读完1天）

1. 这本书的核心观点是什么？（一句话）
2. 最有启发的一个观点是什么？
3. 你实际应用了哪个观点？

答完后我会评判你的记忆准确性。
```

### 方式二：深度复盘（1周后）

用苏格拉底质询检验：

```
📚 复盘：《书名》（读完1周）

5个问题，难度递进：
1. 事实性：作者说了什么？
2. 理解性：为什么这么说？
3. 应用性：怎么用到生活中？
4. 批判性：有什么问题？
5. 迁移性：其他领域适用吗？

评分：X/25
```

### 方式三：应用复盘（1个月后）

检验实际应用：

```
📚 复盘：《书名》（读完1个月）

1. 你实际应用了书中的哪个观点？
2. 效果如何？
3. 遇到了什么问题？
4. 下一步打算怎么做？

应用评分：X/10
```

## 自动提醒

如果用户开启了定时提醒，系统会在：
- 读完1天后：提醒快速复盘
- 读完1周后：提醒深度复盘
- 读完1个月后：提醒应用复盘

## 复盘记录

每次复盘结果记录到 `.reading-review.json`：

```json
{
  "书名": {
    "read_date": "2026-06-16",
    "reviews": [
      {
        "date": "2026-06-17",
        "type": "quick",
        "score": 18,
        "notes": "记忆清晰"
      },
      {
        "date": "2026-06-23",
        "type": "deep",
        "score": 22,
        "notes": "理解深入，但应用不足"
      }
    ]
  }
}
```

## 输出格式

```
📚 复盘结果：《书名》

日期：YYYY-MM-DD
类型：快速/深度/应用
得分：X/25

优势：
- [记得清楚的部分]

薄弱：
- [记不清楚的部分]

建议：
- [下一步行动]
```
