# 知识卡片

## 输出格式

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📚 《书名》知识卡片
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

一句话：[能发朋友圈的核心洞见]

框架：[名称] → [1→2→3→4]

最反直觉：[观点 + 证据]

立即行动：
1. [具体方法]（第X章）
2. [具体方法]（第X章）
3. [具体方法]（第X章）

最大局限：[一句话]

金句："原文"
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 极简版（手机用）

```
📚 《书名》
一句话：[洞见]
框架：[名称]→[步骤]
行动：[1个最具体的方法]
金句："原文"
```

## 严格要求
- 一句话必须精炼到能发朋友圈
- 行动要点必须具体到可以直接做
- 金句必须是原文

$ARGUMENTS
