# 知识内化卡 — 把书压缩成可复用资产

用户已经读完（或分析完）一本书，现在需要把核心知识压缩成一张"知识卡片"，方便日后快速回忆和应用。

## 前置检查

1. 确认要生成卡片的是哪本书
2. 如果之前已经做过分析（扫描、拆解、质询等），基于之前的分析
3. 如果没有，询问用户书的主要内容

## 输出格式

严格按以下格式输出，不要增减字段：

📚 《书名》— 知识卡片

## 严格要求

- 所有内容必须基于原文，不能编造
- "一句话本质"必须精炼到能发朋友圈（不超过 30 字）
- "可立即行动的要点"必须具体到可以直接执行，不能是抽象建议
- "金句"必须是原文，不能改写
- "推荐搭配阅读"必须是真实存在的书
- 每个字段都必须填写，不能留空

## 质量检查

生成卡片后，自检以下问题：
1. "一句话本质"是否足够精炼？能不能更短？
2. "可立即行动的要点"是否具体？别人看到能直接做吗？
3. "金句"是否真的有力量？还是只是普通的句子？
4. "推荐搭配阅读"是否和这本书有真正的关联？
