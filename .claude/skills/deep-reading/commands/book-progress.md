# 阅读进度

## 输出格式

```
📖 阅读进度：《书名》

✅ 全局扫描 [完成]
✅ 跳读标记 [完成]
✅ 第1章拆解 [完成]
✅ 第3章拆解 [完成]
⬜ 第5章拆解 [待做]
⬜ 知识卡片 [待做]

进度：3/5 步 (60%)

下一步：拆解第5章
```

## 进度记录方式

进度记录在项目目录的 `.reading-progress.json` 文件中：

```json
{
  "书名": {
    "started": "2026-06-16",
    "steps": {
      "scan": "done",
      "skip": "done",
      "ch1": "done",
      "ch3": "done",
      "ch5": "pending",
      "card": "pending"
    }
  }
}
```

## 自动恢复

当用户说"继续读《书名》"时：
1. 读取进度文件
2. 从上次中断的地方继续
3. 提示用户已完成和待做的步骤
