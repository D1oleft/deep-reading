# 智能跳读标记

## 标记规则

### 🔴 骨架章（必读）
- 提出全书核心论点的章节
- 引入关键概念框架的章节
- 总结和应用的章节
- 标准：去掉这章，全书论证就不完整

### 🟡 证据章（可略）
- 提供案例、实验、数据支撑的章节
- 重复论证前面已说过的内容
- 标准：略读不影响理解核心论点

### 🟢 延伸章（可跳）
- 讨论相关但非核心的话题
- 作者个人经历/感悟
- 与核心论证关系不大的内容
- 标准：跳过不影响理解本书

## 输出格式

```
📋 《书名》跳读标记

第1章：[标题] 🔴 骨架 — 提出核心框架
第2章：[标题] 🟡 证据 — 案例支撑
第3章：[标题] 🔴 骨架 — 关键概念定义
第4章：[标题] 🟢 延伸 — 个人经历
第5章：[标题] 🔴 骨架 — 核心论证
第6章：[标题] 🟡 证据 — 实验数据
第7章：[标题] 🟢 延伸 — 相关话题

速读建议：只读第1、3、5章（约30分钟）
```

## 标记依据

每个标记必须说明：
- 为什么这章是骨架/证据/延伸
- 引用原文说明这章在论证中的角色
