# 苏格拉底质询

## 出题规则

5个问题，难度递进：
1. 事实性（记住了什么）
2. 理解性（理解了什么）
3. 应用性（能用什么）
4. 批判性（能质疑什么）
5. 迁移性（能迁移什么）

## 评判

用户回答后逐题评判，最后给评分：

```
📊 理解评分：X/25

事实 X/5 | 理解 X/5 | 应用 X/5 | 批判 X/5 | 迁移 X/5

等级：[新手/入门/理解/掌握/精通]
建议：[下一步]
```

## 评分标准
- 1分：记住了表面信息
- 2分：理解了论证逻辑
- 3分：能批判性评价
- 4分：能迁移应用
- 5分：能创造性扩展
