# 跨书连接

## 6个维度

1. **求同**：所有书认同的观点
2. **存异**：分歧点 + 谁更有说服力
3. **互补**：盲区 + 强弱项匹配
4. **辩论**：如果作者们坐在一起
5. **综合**：最佳理解 + 未解决问题
6. **推荐**：下一本读什么

## 输出格式

```
📚 跨书分析：[主题]

共识：[所有书认同的观点]
分歧：[表格对比]
互补：[A的盲区→B的补充]
综合：[拼在一起的最佳理解]
下一本：[推荐 + 理由]
```
