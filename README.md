## ✨ 核心特性

<table>
<tr>
<td width="50%">

## 🚀 快速开始

### 安装（30秒）

```bash
cd ~/.claude/skills
git clone https://github.com/D1oleft/deep-reading.git
```

### 使用

```bash
# 方法1：直接发送内容
"帮我读这本书：[粘贴内容]"

# 方法2：使用命令
/deep-reading:read

# 查看进度
/deep-reading:book-progress

# 导出笔记
/deep-reading:book-export

# 定期复盘
/deep-reading:book-review

# 跨书对比
/deep-reading:book-cross
```

## ⏱️ 时间预估

| 书籍规模 | 预计时间 | 场景 |
|---------|---------|------|
| 📄 小册子（<50页） | 3分钟 | 知乎长文、报告 |
| 📕 中等（50-200页） | 7分钟 | 普通书籍 |
| 📚 大部头（200+页） | 12分钟 | 经典著作 |

## 🆚 为什么选择我们？

| 特性 | Deep Reading | 其他工具 |
|-----|-------------|---------|
| 学习闭环 | ✅ 读→测→复→用 | ❌ 只有分析 |
| 零依赖 | ✅ 纯 Markdown | ❌ 需要 Python/Node |
| 中文优化 | ✅ 原生中文 | ⚠️ 通用适配 |
| 知识卡片 | ✅ Anki 兼容 | ❌ 无 |
| 定期复盘 | ✅ 评分制追踪 | ❌ 无 |
| 费曼检验 | ✅ 讲清楚才算懂 | ❌ 无 |

## 📁 输出示例

### 生成的笔记结构
```
~/笔记/
└── 《君主论》阅读笔记.md
    ├── 📖 书籍信息
    ├── 🎯 核心要点（按章节）
    ├── 🃏 知识卡片（可导入Anki）
    ├── 📊 测验题目与答案
    └── 💭 个人思考
```

### 测验示例
```
Q1: 马基雅维利认为君主应该让人爱还是让人怕？
A. 只能选一个，选怕
B. 最好两者兼备，但必须选一时选怕
C. 爱比怕重要
D. 怕比爱重要，但要避免被恨

答案：D
解析：第17章核心观点——爱与怕可以并存，关键是避免被恨。
```
