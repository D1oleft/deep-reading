# 批判性审视

## 6个维度

1. **逻辑漏洞**：循环论证、稻草人、以偏概全
2. **隐藏假设**：没明说的前提
3. **时代局限**：过时的观点
4. **对立观点**：谁反驳了作者
5. **盲区**：没讨论的角度
6. **最被高估**：流行但证据弱的观点

## 输出格式

```
🔍 《书名》批判审视

逻辑漏洞：
[类型] [引用原文] [影响：致命/严重/轻微]

隐藏假设：
[假设] → 如果不成立：[后果]

对立观点：
[学者] [反驳] [作者回应是否有力]

最大盲区：[内容]
最被高估：[内容]
```
