# 跨书连接

## 功能

把多本书的内容联系起来，找出共同点、分歧点和互补点。

## 6个维度

1. **求同**：所有书认同的观点
2. **存异**：分歧点 + 谁更有说服力
3. **互补**：盲区 + 强弱项匹配
4. **辩论**：如果作者们坐在一起
5. **综合**：最佳理解 + 未解决问题
6. **推荐**：下一本读什么

```

━━━━━━━━━━━━━━━━━━━━
1. 共识
━━━━━━━━━━━━━━━━━━━━
[所有书认同的观点]

━━━━━━━━━━━━━━━━━━━━
2. 分歧
━━━━━━━━━━━━━━━━━━━━

|------|-------|-------|
| 《书A》 | ... | ... |
```
