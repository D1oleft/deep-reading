# 笔记导出

## 功能

把读过的所有内容整理成一个完整的笔记文件，方便复习和分享。

## 导出内容

```
# 《书名》阅读笔记

> 阅读日期：YYYY-MM-DD
> 阅读目的：[用户填写的目的]
> 阅读模式：速读/精读/深度研究

---

## 全局扫描

[扫描结果]

---

## 章节拆解

### 第X章：[标题]

[拆解结果]

### 第Y章：[标题]

[拆解结果]

---

## 知识卡片

[卡片内容]

---

## 批判性审视（如有）

[审视结果]

---

## 跨书连接（如有）

[连接结果]

---

## 阅读感悟

[用户填写，可选]

---

*生成时间：YYYY-MM-DD HH:MM*
```

## 保存位置

默认保存到：`~/笔记/《书名》阅读笔记.md`

用户可自定义路径：
> 笔记保存到哪个目录？（默认：~/笔记/）

## 导出后

```
✅ 笔记已导出

文件：~/笔记/《书名》阅读笔记.md
大小：X KB
包含：全局扫描 + X章拆解 + 知识卡片

要打开查看吗？（回复"查看"或"完成"）
```

## 批量导出

如果用户读过多本书，可以批量导出：

```
📚 已读书籍：

1. 《书名A》— 2026-06-10
2. 《书名B》— 2026-06-15
3. 《书名C》— 2026-06-16

要导出哪些？（回复数字或"全部"）
```

## 用户输入
$ARGUMENTS
