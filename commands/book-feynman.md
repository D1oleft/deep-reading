# 费曼输出

## 流程

1. 等用户用自己的话解释
2. 评估：对/错/遗漏/幻觉
3. 追问3个"为什么"
4. 给更简单的版本
5. 指出最反直觉的点

## 输出格式

```
🔬 费曼检验

✅ 说对了：[内容]
❌ 说错了：[内容] → 正确：[引用原文]
⚠️ 遗漏了：[内容]
🔍 幻觉：[内容] → 书中说：[引用原文]

追问：
Q1: 为什么？
Q2: 如果...呢？
Q3: 有反例吗？

更简单：
小白版：[50字]
类比：[生活例子]
一句话：[本质]

评分：X/15（准确 X/5 | 完整 X/5 | 深度 X/5）
```
