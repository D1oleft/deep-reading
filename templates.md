# 输出模板参考

各命令的标准输出格式说明。

## book-scan 输出结构

```
📖 《书名》全局扫描

1. 核心问题（2-3句话）
2. 一句话论点（不超过30字）
3. 全书逻辑地图（树状结构）
4. 关键概念清单（5-10个，含原文定义）
5. 阅读策略建议（按时间预算分档）
6. 这本书的"敌人"（作者在反驳什么）
```

## book-chapter 输出结构

```
📖 《书名》第X章拆解

1. 核心论点（一句话 + 在全书中的位置）
2. 论证链条（前提→推理→结论，标注来源）
3. 关键概念（定义+边界+反例）
4. 最有价值的3句话（原文 + 价值说明）
5. 隐藏假设（如果不成立会怎样）
6. 与前文关联（承接+矛盾+张力）
7. 一个回答不了的问题
```

## book-quiz 输出结构

```
📊 苏格拉底质询

问题1-5（难度递进：事实→理解→应用→批判→迁移）
用户回答后逐题评判
最后给出理解深度评分（25分制）
```

## book-feynman 输出结构

```
🔬 费曼检验

1. 解释评估（对/错/遗漏/幻觉）
2. 追问3个"为什么"
3. 更简单的版本（小白版+类比版+一句话版）
4. 最反直觉的点
5. 理解评分（15分制）
```

## book-critique 输出结构

```
🔍 批判性审视

1. 逻辑漏洞（具体类型+引用+影响程度）
2. 隐藏假设（如果不成立的后果）
3. 时代局限性（过时观点+新研究）
4. 对立观点（学者+反驳+回应是否有力）
5. 盲区（未讨论的角度）
6. 最被高估的点
+ 辩论模式（可选）
```

## book-card 输出结构

```
📚 知识卡片

一句话本质
核心框架（名称+场景+步骤+来源）
三个最反直觉的观点
三个可立即行动的要点
最大的局限
推荐搭配阅读
金句收藏（3句原文）
```

## book-cross 输出结构

```
📚 跨书综合分析

1. 共识（所有书认同的观点）
2. 分歧（表格对比+谁更有说服力）
3. 互补（盲区+强弱项匹配）
4. 作者辩论（想象场景）
5. 综合理解（最佳理解+未解决问题）
6. 推荐下一本
```
